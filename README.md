# clockServer
## about
プロ生ちゃん時報サーバー(仮)の後を継ぐ時計サーバー．
多分raspberry pi Bにraspbianで動かす．

## setup
1. run `bundle install`
2. write your alerm time on "time.yml"
3. locate your wav file at "wav/<time>/<name>.wav"
